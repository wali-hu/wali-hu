## Who I Am

I build on **Solana** and **EVM** — not because it's trending, but because low-level blockchain programming genuinely fascinates me.

Started from Cyfrin Updraft. Learned Solidity, Foundry, shipped real client contracts. Then Solana and native Rust changed everything. Rust clicked in a way few things do. I went from Anchor dApps → writing raw on-chain programs without a framework — manual PDAs, versioned transactions, ALTs.

Currently at **ProgrammX** shipping production deliverables across both ecosystems. Self-motivated, comfortable with async workflows and distributed teams. Long-term direction: **protocol engineering** — consensus mechanisms, DeFi primitive design, MEV dynamics, runtime execution.
