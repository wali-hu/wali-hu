## Who I Am

I build on **Solana** and **EVM** — not because it's trending, but because low-level blockchain programming genuinely fascinates me.

Started from Cyfrin Updraft. Learned Solidity, Foundry, shipped real client contracts. Then Solana and native Rust changed everything. Rust clicked in a way few things do. I went from Anchor dApps → raw on-chain programs, manual PDAs, versioned transactions, ALTs — no framework, just the metal.

Currently at **ProgrammX**, shipping production deliverables on both ecosystems. Long-term direction: **protocol engineering**.
